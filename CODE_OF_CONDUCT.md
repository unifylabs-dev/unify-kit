# Code of Conduct

This project adopts the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)
as its code of conduct. The full canonical text — including the pledge, the
standards we hold ourselves to, and the enforcement guidelines — lives at that
URL and is incorporated here by reference.

## Scope

The Contributor Covenant applies in all project spaces — this repository's
issues, pull requests, discussions, code review threads, and any project-
adjacent channels — and when an individual is officially representing the
project in public spaces.

## Reporting

Concerns about conduct in this project's spaces are reported through the same
private channel as security issues, via GitHub's **Report a vulnerability**
flow on the repository's **Security** tab (see [`SECURITY.md`](SECURITY.md)).

This routes the report to the project maintainer privately. Include:

- What happened, and where (link to the issue / PR / thread if applicable)
- When it happened
- Whether the conduct is ongoing

If the reporter would prefer to remain anonymous to the maintainer as well as
the public, say so in the report.

## Response

The maintainer acknowledges conduct reports within **5 business days** of
receipt and responds with the action taken (or the reasoning, if no action is
taken) within 10 business days.

Enforcement follows the **Enforcement Guidelines** section of Contributor
Covenant v2.1 — a four-step escalation from private correction through
temporary ban to permanent ban, with the severity matched to the impact of the
behavior.
